# ConnectKind – Community Skill & Resource Exchange Platform

This project provides a web-based platform that enables users to share skills and resources locally without monetary transactions.

---

## Architecture Diagram

![Architecture Diagram](architecture.png)

---

## Components

### Frontend

- index.html
- login.html
- profile.html
- add-listing.html
- search.html
- chat.html

Provides:
- User interface
- Form handling
- Real-time updates
- Chat functionality

### Backend (Firebase – Serverless)

- **Firebase Authentication** – Secure login and registration.
- **Firebase Firestore** – Stores users, listings, and chat messages.
- **Firebase Hosting** – Deploys the application.

---

## Usage

1. Open the Live Demo link.
2. Register or log in.
3. Create your profile.
4. Add a listing.
5. Search for listings.
6. Start real-time chat.
